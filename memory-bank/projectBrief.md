# Project Brief

## Project Name
StitchPoint (branded as "Naqsh Couture")

## Purpose
Full-stack e-commerce platform for a fashion/clothing marketplace connecting customers with store managers/artisans selling clothing products. Focused on Pakistani fashion market (PKR currency, references to Karachi, Lahore, Sindh).

## Target Users
- **Customers**: Browse products, shop by categories, manage wishlist/cart, checkout, track orders, write reviews
- **Store Managers/Artisans**: Manage products, process orders, view analytics, respond to reviews
- **Super Admins**: Platform oversight, user management, commission management, system settings

## Key Differentiators
- Multi-role system with role-specific dashboards
- Tiered commission system (5%/8%/10% based on product price)
- Store-level product management with district-based filtering
- Built-in analytics for managers and admins
