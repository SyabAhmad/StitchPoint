# Product Context

## Overview
StitchPoint is a full-stack e-commerce platform built with React 19 frontend and Flask/PostgreSQL backend. The platform serves as a marketplace for Pakistani fashion, connecting customers with clothing store managers and artisans.

## Core Features

### Customer Features
- Browse products with filtering (category, price, district)
- Shop by categories and collections
- Product detail pages with reviews
- Wishlist management
- Shopping cart and checkout flow
- Order tracking and history
- Review/rating system

### Store Manager Features
- Dedicated manager dashboard
- Product CRUD with image uploads
- Order management and status updates
- Sales analytics (Recharts)
- Review and comment moderation
- Category management

### Super Admin Features
- Platform-wide analytics
- User management (customers, managers)
- Store analytics across all stores
- Commission rate management
- System settings

## Technical Stack

### Frontend
- React 19 with React Router DOM 7
- Vite 7 (build tool with SWC)
- Tailwind CSS 4
- Recharts (analytics)
- React Leaflet (store maps)
- React Hot Toast (notifications)
- Three.js (3D graphics)

### Backend
- Python Flask with Flask-SQLAlchemy ORM
- PostgreSQL database
- Flask-JWT-Extended (authentication)
- Flask-Migrate (database migrations)
- Flask-CORS (cross-origin support)
